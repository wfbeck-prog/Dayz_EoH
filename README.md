# Dayz_EoH

Echoes of Humanity DayZ server/mod workspace.

This repository is set up as a structured workspace for your EoH DayZ projects. Fill in the folders with the files that are already working on your server.

## Recommended workflow

1. Keep `working/` for files that are currently live or confirmed working.
2. Keep `mods/` for packaged mod projects.
3. Keep `mission/` for mission/server-side files.
4. Keep `docs/` for setup notes, install steps, classnames, and troubleshooting.
5. Keep `archive/` for old systems you may want to reference but should not load.

## Main goals

- Consolidate EoH systems over time into a clean `EoH_Server` mod.
- Preserve working files before refactoring.
- Separate experimental code from live server code.
- Track changes safely with GitHub history.
